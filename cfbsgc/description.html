An Interactive Tool for Exploring Fragment Library Principle Component Analysis (PCA)

- Check the boxes to add new libraries to the scatter plots.
- Use the dropdown menus to select descriptors to scale circle color, size, and opacity.
- Use the "Principle Component Analysis" dropdown menu to select the molecular descriptor calculation to use in PCA analysis.
- Hover over the circles to view chemical structures and descriptor values associated with circle color, size, and opacity.
- Use the tools to the right of the scatter plot to pan, zoom, select, refresh and save the active plots.
* References for selected descriptors are located at the bottom of the page.
** Chemical descriptors were calculated using the RDKit python package (http://www.rdkit.org/).
